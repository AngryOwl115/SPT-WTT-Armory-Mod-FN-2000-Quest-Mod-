# SPT-WTT-Mod-FN-2000-Quest-Mod-
A custom Peacekeeper quest for SPT Tarkov's WTT mod featuring the FN F2000. I made this mod because the original WTT mod doesnt seem to have a quest surrounding this amazing weapon. And so I made it for myself! Figured Id share! This mod will add a simple, yet interesting quest revolving the weapon, the Labyrinth and some lore. I hope you like it!
