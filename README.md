# SPT-WTT-Armory-FN-2000-Quest-Mod-

A custom Peacekeeper quest for SPT Tarkov's WTT-Armory mod featuring the FN F2000. I made this custom quest because the original WTT Armory mod doesnt seem to have a dedicated quest surrounding this amazing weapon. And so I made it for myself! Figured Id share it to others who might appreciate it! 
This mod will by default make the original FN2000 barter be locked so that you have to do this quest to unlock it and add a simple, yet interesting n challenging quest revolving around the Labyrinth and some lore that I came up with the weapon so that you can find and unlock it! I hope you like it!
Mainly made this mod because I really love the F2000 and I wanted it to have some more love by having its own quest! Hope yall like it!
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Download Instructions - 
Must have WTT's Armory mod installed https://forge.sp-tarkov.com/mod/2246/wtt-armory
Then all you need to do is download the ZIP folder and drag the SPT folder to your SPT directory folder! Replace files when the option comes up.
