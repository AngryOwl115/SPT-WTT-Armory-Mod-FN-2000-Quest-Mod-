# SPT-WTT-Arsenal-Mod-FN-2000-Quest-Mod-

A custom Peacekeeper quest for SPT Tarkov's WTT mod featuring the FN F2000. I made this mod because the original WTT mod doesnt seem to have a quest surrounding this amazing weapon. And so I made it for myself! Figured Id share! This mod will add a simple, yet interesting quest revolving the weapon, the Labyrinth and some lore that I came up with. I hope you like it!
Mainly made this mod because I really love the F2000 and I wanted it to have some more love by having its own quest! Hope yall like it!
Download Instructions - 
Must have WTT's Armory mod installed https://forge.sp-tarkov.com/mod/2246/wtt-armory
Then all you need to do is download the ZIP folder and drag the SPT folder to your SPT directory folder! Replace files when the option comes up.
